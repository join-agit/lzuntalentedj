# 随心秀(react版h5微场景编辑器)

## 目标
> 打造一款现象级微场景编辑器 [演示地址](http://show.lzuntalented.cn)

## 使用

```bash
npm i
// 开发
npm start
// 浏览 http://localhost:9901

// 构建
npm run build
```
